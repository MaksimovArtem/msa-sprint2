## values-v1.yaml

Версия v1 и без флага.
Service вклчюается
```
replicaCount: 1
appName: booking-service
version: v1

image:
  name: booking-service
  tag: latest
  pullPolicy: Never

service:
  enabled: true
  type: ClusterIP
  port: 80
  targetPort: 8080

resources:
  enableFeatureX:
    value: false
  k8s:
    requests:
      cpu: 50m
	  memory: 64Mi
	limits:
	  cpu: 200m
	  memory: 128Mi

livenessProbe:
  initialDelaySeconds: 5
  periodSeconds: 10
  timeoutSeconds: 1
  failureThreshold: 3

readinessProbe:
  initialDelaySeconds: 2
  periodSeconds: 5
  timeoutSeconds: 1
  failureThreshold: 3
```
